## Requirements

- Python 3.12 or later
- .NET 9 or later
- Docker
- protoc
- uv
- Visual Studio Code (optional)

## General steps to start from scratch

1. Create .NET and Python projects

1. Configure dependencies
1. Generate messages from protobufs
1. Define agents
